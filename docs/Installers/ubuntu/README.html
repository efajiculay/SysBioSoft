BioSANS installer for Ubuntu

	Main executable installer:
	
	1) BioSANS_Ubuntu_Installer - we provide an executable to install BioSANS but
	   still requires the terminal.
	   
	To install:
	
	   1) Download  BioSANS_Ubuntu_Installer
	   2) Launch the Ubuntu Terminal using Ctrl + t
	   2) cd to the directory of BioSANS_Ubuntu_Installer
	   3) run the following command

			./Ubuntu_Installer
		
	   4) Key in the Ubuntu password, press enter or type Y for the steps where 
	      the OS asked them as the installation proceeds.
	   
	      Files BioSANS.sh and BioSANS.desktop will appear on the desktop after
	      the installation.
		  
	   5) Right click the BioSANS.desktop file and ticked the allow launching.
	   
	To run BioSANS:
	
          Double click the desktop shortcut to launch BioSANS.
		  
          Type BioSANS.sh in terminal and hit enter

	BioSANS can be launche din terminal as well by any of the following;
	
		  BioSANS 
		  python3 -m BioSANS2020
		  python3 -m BioSANS2020.BioSANS 
		  
		  or 
		  
		  BioSSL 
		  python3 -m BioSANS2020.BioSSL

Installation using pip

	
	Installation using pip is thw preferred way in Ubuntu since python 3.8 or higher 
	is normally avialable in Ubuntu operating system.
	
	We can simply Install BioSANS by running the following command in Ubuntu Terminal;  

	For anaconda users:

		Create new conda envinroment where you can install BioSANS

			conda create -n BioSANS python==3.8.8
		   
		Activate conda environment

			activate BioSANS
		   
		Install BioSANS using pip

			pip install BioSANS2020 --upgrade

	For non anaconda users:

			new way:

			pip install BioSANS2020 --upgrade
			
	After the installation, you can launcehd BioSANS GUI by any of the following
	terminal commands;
	   
		BioSANS 
		python3 -m BioSANS2020
		python3 -m BioSANS2020.BioSANS 
	   
	To launched BioSSL which is a command line structured simulation langguage
	associated with BioSANS; Issue any of the following commands;
	   
		BioSSL 
		python3 -m BioSANS2020.BioSSL

	To uninstall BioSANS, issue the following command

	   pip uninstall BioSANS2020

If the above installation procedure fails

	   
		#The following script are optional
		sudo apt update
		sudo apt install software-properties-common
		#sudo add-apt-repository ppa:deadsnakes/ppa   -might not be needed in most cases

		#If python3.7 or higher is not installed (normally already have 3.8 by default)
		sudo apt install python3
		sudo apt-get install python3-tk
		sudo apt install python3-distutils

		#Installing pip here
		sudo apt install curl
		curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
		python3 get-pip.py

		#Installing BioSANS
		python3 -m pip install BioSANS2020 --upgrade

		#Locating BioSANS installation directory
		python3 -m pip show BioSANS2020

		# to run BioSANS
		cd "to the location directory"
		python3 BioSANS.py

		#You may also run BioSANS using any of the following commands
		BioSANS
		python3 -m BioSANS2020
		python3 -m BioSANS2020.BioSANS
