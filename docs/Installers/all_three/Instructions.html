BioSANS installation procedure

	
	BioSANS installation  can easily be done if anaconda  or python terminal is 
	already installed in the user machine. The following link is where anaconda
	and python can be downloaded;
	
			https://www.anaconda.com/products/individual
			
			https://www.python.org/downloads/
	
	Download and  install one of those software and ensure that you have ticked
	the option to add them to path.
	
	After installing anaconda and or python from the corresponding links above,
	The user needs to find out how to lauched the terminal associated to either
	python or anaconda. 
	
	In windows, you  can type cmd or anaconda in the windows search  located on
	the left side of the Windows taskbar.
	
	In Ubuntu, just use CTRL + T to launched the terminal. Normally, python is
	already installed in Ubuntu and probably dont need to install any of the 
	links above.
	
	For MacOSX,  click the Launchpad. Search  for the  terminal and open it. If 
	you  have ticked  the option to add python/anaconda  to path, then they are 
	accessible in MacOSX terminal.
	
	In the terminal just type the following to install BioSANS;
	
			pip install BioSANS2020 --upgrade

	To run BioSANS; just need to run any of this command in the same terminal;
	
			BioSANS 
			python -m BioSANS2020
			python -m BioSANS2020.BioSANS
			
	To run BioSSL; just need to run any of this command in the same terminal;
	
			BioSSL 
			python -m BioSANS2020.BioSSL
			
	To uninstall; run the following commands;
	
			python -m pip uninstall BioSANS2020
			
	To see BioSANS installation directory; type the followung
	
			python -m pip show BioSANS2020

Windows intallation steps

	
	
	Main installers:
	
	Executable installer(Double click executable installer)

	1) BioSANS_installer_py3.9.5_amd32.exe   - for windows > 7
	2) BioSANS_installer_py3.9.5_amd64.exe   - for windows > 7
	3) BioSANS_installer_py3.7.4_x86_64.exe  - for windows > XP
	4) BioSANS_installer_py3.7.4_x86_32.exe  - for windows > XP
	
	Installation using pip - preferred way(python/anaconda terminal)
	
		for windows version > XP where python >= 3.7 is installed  

	For anaconda terminal users:

		Create new conda envinroment where you can install BioSANS

			conda create -n BioSANS python==3.9
		   
		Activate conda environment

			activate BioSANS
		   
		Install BioSANS using pip

			pip install BioSANS2020 --upgrade

	For python terminal users:

			pip install BioSANS2020 --upgrade
			
	After the installation, you can launcehd BioSANS GUI by any of the following terminal commands;
	   
		BioSANS 
		python3 -m BioSANS2020
		python3 -m BioSANS2020.BioSANS 
	   
	To launched BioSSL which is a command line structured simulation langguage associated with BioSANS;
	Issue any of the following commands;
	   
		BioSSL 
		python3 -m BioSANS2020.BioSSL

	To uninstall BioSANS, issue the following command

	   pip uninstall BioSANS2020

	   
	### Uninstallation procedure:

	For installers 1-4, Uninstall the corresponding python version in the control panel 

	For installed using pip, type the following command.

		   pip uninstall BioSANS2020

Ubuntu intallation steps

	

	### Main installers:
	
	1) Installation using pip - preferred way(Ubuntu terminal)
	
	In most cases, Ubuntu comes with python and recently python 3.8 is default in Ubuntu.
	We can simply Install BioSANS by running the following command in Ubuntu Terminal;  

	For anaconda users:

		Create new conda envinroment where you can install BioSANS

			conda create -n BioSANS python==3.9
		   
		Activate conda environment

			activate BioSANS
		   
		Install BioSANS using pip

			pip install BioSANS2020 --upgrade

	For non anaconda users:

			pip install BioSANS2020 --upgrade
			
	After the installation, you can launcehd BioSANS GUI by any of the following terminal commands;
	   
		BioSANS 
		python3 -m BioSANS2020
		python3 -m BioSANS2020.BioSANS 
	   
	To launched BioSSL which is a command line structured simulation langguage associated with BioSANS;
	Issue any of the following commands;
	   
		BioSSL 
		python3 -m BioSANS2020.BioSSL

	To uninstall BioSANS, issue the following command

	   pip uninstall BioSANS2020
   
		   
	2) Installation using executable - we provide an executable to install BioSANS but still requires the terminal.

		BioSANS_Ubuntu_Installer - just cd to the directory of this executable after you download it and type;

		./Ubuntu_Installer
	   
	   This will also create a desktop shortcut to run BioSANS
	   
	   
	3) If the above installation procedure fails
	
		#The following script are optional
		sudo apt update
		sudo apt install software-properties-common
		#sudo add-apt-repository ppa:deadsnakes/ppa   -might not be needed in most cases

		#If python3.7 or higher is not installed (normally already have 3.8 by default)
		sudo apt install python3
		sudo apt-get install python3-tk
		sudo apt install python3-distutils

		#Installing pip here
		sudo apt install curl
		curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
		python3 get-pip.py

		#Installing BioSANS
		python3 -m pip install BioSANS2020 --upgrade

		#Locating BioSANS installation directory
		python3 -m pip show BioSANS2020

		# to run BioSANS
		cd "to the location directory"
		python3 BioSANS.py

		#You may also run BioSANS using any of the following commands
		BioSANS
		python3 -m BioSANS2020
		python3 -m BioSANS2020.BioSANS
